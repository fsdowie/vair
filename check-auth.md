# Check Supabase Auth Settings

Go to: https://supabase.com/dashboard/project/iunehbdazfzgfclkvvgd/auth/providers

Make sure:
1. Email provider is ENABLED
2. "Confirm email" is DISABLED (for testing)
3. "Secure email change" is DISABLED (for testing)

If these are not set correctly, update them and try signing up again with a new email.
